# 图片资源文件夹

请将以下图片文件放入此文件夹：

## 背景图片
- `bg.jpg` - 首页初始背景图片
- `bg2.jpg` - 点击"开始探索"后更换的背景图片

## 第一章照片
- `photo1.jpg` - 第一张照片
- `photo2.jpg` - 第二张照片  
- `photo3.jpg` - 第三张照片
- `photo4.jpg` - 第四张照片（显示最终文字的照片）

## 图片要求
- 建议尺寸：1920x1080 或更高分辨率
- 格式：JPG 或 PNG
- 文件大小：建议每张图片不超过 2MB

## 注意事项
- 确保图片文件名与HTML中的引用完全一致
- 图片质量会影响网站的整体视觉效果
- 建议使用高质量、主题相关的图片
