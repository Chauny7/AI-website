嵌入功能使用示例

注意：这些示例需要在主页面（index.html）中运行，因为它们依赖于主页面的JavaScript函数。

1. 嵌入图片示例

点击下面的按钮来嵌入不同的图片：

// 代码示例 embedImageByCode('https://picsum.photos/600/400?random=1', '随机图片1');

2. 嵌入Tableau示例

点击下面的按钮来嵌入Tableau图表：

// 代码示例 embedTableauByCode('https://public.tableau.com/views/RegionalSampleWorkbook/Storms');

3. 返回图表模式

点击下面的按钮返回默认的图表显示：

// 代码示例 showChart();

4. 完整使用流程

这是一个完整的使用流程示例：

// 完整流程示例 function demoWorkflow() { // 1. 嵌入图片 embedImageByCode('https://picsum.photos/600/400?random=4', '演示图片'); // 2. 3秒后嵌入Tableau setTimeout(() => { embedTableauByCode('https://public.tableau.com/views/RegionalSampleWorkbook/Storms'); }, 3000); // 3. 6秒后返回图表 setTimeout(() => { showChart(); }, 6000); }

5. 自定义图片路径

你可以使用本地图片或网络图片：

// 本地图片示例 embedImageByCode('assets/chapter1（1）.png', '本地图片'); // 网络图片示例 embedImageByCode('https://via.placeholder.com/600x400/007bff/ffffff?text=自定义图片', '占位图片');
